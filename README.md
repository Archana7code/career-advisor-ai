## 📖 Overview

**SkillPop** is an AI-powered career guidance platform that brings together multiple career development tools into a single, responsive web application.

The platform enables users to generate personalized learning roadmaps, receive AI-powered resume feedback, practice mock interviews, interact with an AI career assistant, explore curated learning resources, and stay consistent with daily coding challenges.

The goal of SkillPop is to simplify career preparation by providing practical guidance and interactive AI tools in one place.

# ✨ Features

- 🤖 AI Career Chatbot for career-related guidance
- 📄 AI Resume Analyzer with personalized feedback
- 🗺️ Personalized Learning Roadmap Generator
- 🎤 AI-Powered Mock Interview Assistant
- 📚 Curated Courses & Career Opportunities
- 💻 Daily Coding Challenges
- 🌙 Dark / Light Theme

# 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Archana7code/career-advisor-ai.git
```

Move into the project directory

```bash
cd career-advisor-ai
```

Install dependencies

```bash
npm install
```

Create a `.env` file using the `.env.example` template and configure the required API keys.

Run the development server.

---

# 🔑 Environment Variables

Create a `.env` file in the project root.

Example

```env
GEMINI_API_KEY=YOUR_API_KEY
```

---

# ⚠️ Important

This repository does **not** include API keys or sensitive credentials.

To use the AI-powered features, create a `.env` file and add your own API credentials.
